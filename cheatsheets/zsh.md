# Zsh cheat sheet

## Native

| keymap | description |
|--------|-------------|
| ctrl+l | Clear screen |
| ctrl+p | Select previous command |
| ctrl+n | Select next command |
| ctrl+a | Move to the start of the line |
| ctrl+e | Move to the end of the line |
| ctrl+f | Skip word forwards |
| ctrl+b | Skip word backwards |
| ctrl+w | Delete word |
