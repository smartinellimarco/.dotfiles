# Kitty cheat sheet

## MacOS

| keymap | description |
|--------|-------------|
| ⌘+t | Open new tab |
| ⌘+\<number> | Go to tab \<number> |
| ⌃+⇥ | Next tab |
| ⌃+⇧+⇥ | Previous tab |
| ⌘+w | Close tab |
| ⌘+q | Quit application |
| ⌘+c | Copy to clipboard |
| ⌘+v | Paste from clipboard |
